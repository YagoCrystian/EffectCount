# RPG Effect Tracker

Ferramenta simples para rastrear **buffs, debuffs e efeitos de controle** em personagens durante combates de **RPG de mesa**.

O objetivo é facilitar o acompanhamento da **duração dos efeitos ao longo dos turnos**, evitando anotações manuais ou esquecimentos durante a sessão.

---

# 🎯 Intuito do Projeto

Durante combates em RPG de mesa é comum existir vários efeitos ativos, como:

- Atordoado
- Envenenado
- Lentidão
- Buffs mágicos
- Debuffs
- Efeitos de controle com duração por turno

Com vários personagens em combate, acompanhar manualmente esses efeitos pode ficar confuso.

Este tracker foi criado para:

- Organizar efeitos por personagem
- Controlar duração de cada efeito
- Reduzir automaticamente a duração ao passar o turno
- Facilitar a visualização de todos os efeitos ativos

---

# 🚀 Como Usar

1. Abra o site no navegador.

2. Digite o **nome do personagem** e clique em **Adicionar**.

3. Clique em **Efeito** para adicionar um buff ou debuff.

4. Informe:
   - Nome do efeito
   - Duração inicial

5. Use os botões para controlar os efeitos:

**+** → aumenta a duração  
**-** → diminui a duração  
🗑 → remove o efeito  

6. Clique em **Passar Turno** para reduzir a duração de todos os efeitos ativos.

7. Use **Resetar Tudo** para limpar todos os personagens e efeitos.

---

# ⚙️ Funcionalidades

- Adicionar personagens
- Editar nome do personagem
- Remover personagens
- Adicionar buffs e debuffs
- Controlar duração manualmente
- Redução automática ao passar turno
- Remoção automática quando a duração chega a 0
- Interface simples para uso rápido durante combate
